# Gagan C B — Immersive Digital Portfolio

A premium, interactive, and high-end digital portfolio showcasing engineering accomplishments, projects, and certifications. Designed with a dark luxury aesthetic, typography-driven layouts, and custom-crafted micro-interactions.

---

## 🌟 Key Features

- **Constellation Background**: An interactive HTML5 Canvas background that responds to mouse movements.
- **Custom Adaptive Cursor**: A custom dual-state cursor (dot and outer ring) that expands and morphs on interactive elements.
- **Dynamic Loader**: A smooth loading screen with custom progress percentage computation.
- **Auto-Typing & Scroll Reveal**: Smooth typing effect for key roles and scroll-triggered fade-in animations for section items.
- **Interactive Skill System**: CSS-driven skill bars with scroll-triggered animation and interactive hover states.
- **Mockup Tilt Parallax**: Interactive 3D tilt effects on project mockups using mouse vector calculations.
- **Document Hub**: Fully integrated certificates and resumes hosted locally inside the `doc` folder.
